# Landing Page --- Plataforma de Microaprendizagem

# Objetivo

Esta landing page deve vender a transformação, não a tecnologia.

O visitante **não está procurando uma IA**, ele está procurando uma
maneira de aprender mais, evoluir profissionalmente e finalmente
conseguir absorver o conhecimento dos livros que sempre quis ler.

A comunicação deve focar na dor, no desejo e no resultado.

# Seção 5 --- Como funciona

## 1. Escolhemos grandes livros

Obras cuidadosamente selecionadas sobre negócios, produtividade,
desenvolvimento pessoal, liderança, psicologia, comunicação, tecnologia
e muito mais.

## 2. Estudamos a obra completa

Cada livro é analisado integralmente para preservar o contexto, os
conceitos e a mensagem do autor.

Você recebe uma leitura de aproximadamente 20 minutos contendo:

-   principais conceitos;
-   ideias centrais;
-   exemplos;
-   aplicações práticas;
-   trechos importantes da obra;
-   perguntas para reflexão.

## 4. Receba automaticamente

Escolha receber por:

-   E-mail

# Direção Criativa e Design da Landing Page

## Objetivo do Design

O design não deve transmitir a sensação de uma plataforma de cursos ou
de um SaaS comum.

A experiência precisa comunicar imediatamente três ideias:

-   conhecimento de alto valor;
-   sofisticação;
-   simplicidade.

O visitante deve sentir que está entrando em uma biblioteca moderna,
construída para pessoas que valorizam aprendizado contínuo.

O visual precisa gerar a percepção de que aprender pode ser uma
experiência elegante, prazerosa e inspiradora.

------------------------------------------------------------------------

# Conceito Visual

A identidade visual deve seguir um conceito minimalista premium.

Evite excesso de elementos, gradientes exagerados, muitas cores ou
animações desnecessárias.

A página deve transmitir calma, inteligência e confiança.

A sensação desejada é semelhante a produtos como:

-   Apple
-   Notion
-   Linear
-   Raycast
-   Stripe
-   Headway
-   Readwise

Poucos elementos, muito espaço em branco e excelente tipografia.

------------------------------------------------------------------------

# Paleta de cores

## Cor principal

Um azul profundo ou verde petróleo, transmitindo confiança e
concentração.

## Cor de destaque

Um tom quente (âmbar ou dourado suave) apenas para CTAs e pequenos
detalhes.

## Fundo

Off-white (#FAFAF8) ou cinza muito claro.

## Texto

Cinza quase preto, priorizando legibilidade.

------------------------------------------------------------------------

# Tipografia

A tipografia é um dos elementos mais importantes da identidade.

Utilizar fontes modernas e elegantes.

Sugestões:

-   Inter
-   Manrope
-   Plus Jakarta Sans
-   General Sans

Os títulos devem ser grandes, leves e com bastante espaçamento.

Evite blocos enormes de texto. A leitura deve parecer leve.

------------------------------------------------------------------------

# Layout

A página deve utilizar uma largura confortável, bastante espaço negativo
e ritmo visual.

Cada seção deve "respirar".

Evite cartões excessivos.

Prefira:

-   divisões por blocos;
-   ilustrações discretas;
-   mockups grandes;
-   títulos impactantes.

------------------------------------------------------------------------

# Ilustrações

Priorizar:

-   mockups realistas;
-   telas da plataforma;
-   mensagens de WhatsApp;
-   visualização do e-mail.

Evitar:

-   imagens genéricas de pessoas felizes;
-   bancos de imagens artificiais;
-   excesso de ícones coloridos.

------------------------------------------------------------------------

# Microinterações

Adicionar pequenas animações para aumentar a percepção de qualidade:

-   fade-in suave ao rolar;
-   cartões elevando ao passar o mouse;
-   botões com transições discretas;
-   contador de estatísticas;
-   barra de progresso da leitura.

Nunca utilizar animações chamativas que distraiam do conteúdo.

------------------------------------------------------------------------

# Experiência Mobile

A experiência deve ser pensada primeiro para dispositivos móveis.

Grande parte dos visitantes conhecerá a plataforma pelo celular.

Os CTAs devem permanecer sempre acessíveis.

Os mockups devem ocupar boa parte da tela para reforçar o valor do
produto.

------------------------------------------------------------------------

# Tom Visual

Se o produto pudesse ser resumido em palavras, seriam:

-   Inteligente
-   Elegante
-   Minimalista
-   Inspirador
-   Sofisticado
-   Atemporal

O usuário deve sentir que investir alguns minutos por dia em aprendizado
é uma decisão inteligente, e o design precisa reforçar essa percepção em
cada detalhe.

------------------------------------------------------------------------

# Princípio de Design

A interface deve parecer uma extensão do próprio hábito de leitura:

menos distrações, mais foco.

Cada elemento visual deve existir para aproximar o usuário do
conhecimento, nunca para competir com ele.

A sensação final deve ser:

> "Esta plataforma foi feita para quem leva o aprendizado a sério."
